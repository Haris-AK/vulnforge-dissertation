# CM3203 initial plan formative feedback framework

Use this file for **formative** feedback on CM3203 initial plan drafts only. It is not a marking scheme and must not be used for mark prediction.

## Criteria

Use these exact headings:

- **Project Description**
- **Aims and Objectives**
- **Work Plan, Feasibility and Risk Management**
- **Communication Skills**

Note: the current guide uses **Work Plan** as the section name, although one assessment-scale line may still refer to a "time plan section". Prefer **Work Plan** in student-facing advice.

## Criterion guidance

### Project Description

Consider:
- whether the title and description accurately represent the project
- whether the project is suitable for CM3203 and the scope is realistic for one semester
- whether the description is project-specific rather than generic

Typical improvement points:
- make the title more specific and technically informative
- sharpen the description so the exact problem and intended output are clear
- narrow or rebalance scope to fit the module
- make expected deliverables more explicit

### Aims and Objectives

Consider:
- whether aims are clear and technically meaningful
- whether objectives are specific, actionable, and appropriately challenging
- whether objectives support a coherent final report and deliverables later on

Typical improvement points:
- convert broad intentions into concrete objectives
- distinguish core objectives from stretch goals
- make objectives testable or assessable where possible
- ensure at least one objective addresses validation or evaluation

### Work Plan, Feasibility and Risk Management

Consider:
- whether the work plan covers the required period (e.g. spring weeks 1–12)
- whether tasks, milestones, and deliverables are clearly linked
- whether sequencing is realistic and risks/dependencies are addressed

Typical improvement points:
- link tasks to milestones and deliverables explicitly
- identify high-risk dependencies and add mitigations
- include contingency or re-planning time where appropriate
- balance background work, implementation, and evaluation preparation

### Communication Skills

Consider:
- whether the plan is clearly structured and the required sections are present
- whether the writing is concise and understandable
- whether tables or lists support clarity where appropriate

Typical improvement points:
- tighten wording and remove vague claims
- use clearer headings and signposting
- make the plan easier to scan; present milestones in a readable way
